# TaskConnect – Hyperlocal Service Marketplace

## Overview
TaskConnect is a cross-platform mobile application built using Flutter that connects users with trusted local helpers for everyday tasks such as home services, errands, and on-demand assistance.

The platform focuses on simplicity, reliability, and real-world usability while enabling flexible earning opportunities for service providers.

## Features
- User authentication  
- Task posting and management  
- Real-time updates  
- Clean and responsive UI  
- Scalable backend integration  

## Tech Stack
- Flutter (Frontend)
- Dart
- Firebase / Node.js (Backend)
- REST APIs
